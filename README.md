# AI Prompt Library / AIプロンプトライブラリー

AI生成のためのプロンプトコレクションサイトです。

## 機能

- **100件の高品質プロンプト**: 文章生成、画像生成、コーディング、音声・音楽、動画の5カテゴリ
- **キーワード検索**: タイトル、本文、タグから検索可能
- **カテゴリ・タグフィルタリング**: 2段階の階層フィルタリング
- **困っていることから探す**: お悩みを入力するとおすすめプロンプトを3件提案
- **ワンクリックコピー**: プロンプトをクリップボードにコピー
- **推奨ツールリンク**: 各プロンプトに適したAIツールへのリンク

## ローカル開発

```bash
# ローカルサーバーを起動
python -m http.server 8000

# ブラウザでアクセス
# http://localhost:8000
```

## データ管理

プロンプトデータは `prompts.csv` で管理しています。

```bash
# CSVを編集後、JSONに変換
python convert.py
```

## ファイル構成

```
ai_prompt_library/
├── index.html      # メインHTML
├── style.css       # スタイルシート
├── script.js       # JavaScript
├── data.json       # プロンプトデータ（自動生成）
├── prompts.csv     # プロンプトデータ（編集用）
├── convert.py      # CSV→JSON変換スクリプト
└── README.md       # このファイル
```
